## 1. AI 서비스 명칭

**TryAngle** (트라이앵글)

- "Try (시도하다)" + "Angle (카메라 앵글)" 의 합성어
- 누구나 쉽게 전문가 수준의 사진을 찍을 수 있도록 돕는 AI 촬영 가이드 앱

## 3. 핵심 내용

### 서비스 개요

TryAngle은 사진 촬영 초보자도 전문가처럼 찍을 수 있도록 **AI가 실시간으로 가이드**하는 모바일 애플리케이션입니다.

**현재 SNS 트렌드 반영:**
- 사진학적 구도뿐만 아니라 '감성'을 중시하는 현대 트렌드 반영
- 사용자가 원하는 레퍼런스 사진을 선택하면, AI가 실시간으로 비교 분석
- "왼팔을 15도 올려주세요", "2걸음 뒤로 가세요" 등 **구체적 수치 피드백**

### 주요 기능

#### (1) 실시간 AI 코칭 ⭐ (핵심 기능)

**기술 구현 현황:**
- ✅ YOLO11s + RTMPose 통합 완료
- ✅ 실시간 10fps 분석 (150-200ms 처리 속도)
- ✅ 히스테리시스 필터 적용 (10프레임 연속 감지 후 표시)

**사용자 경험:**
```
레퍼런스 선택
    ↓
실시간 카메라 프레임 분석 (10fps)
    ↓
구체적 피드백 제공:
  - "왼팔을 15도 더 올려주세요"
  - "오른쪽으로 2걸음 이동하세요"
  - "고개를 왼쪽으로 10도 돌려주세요"
    ↓
완성도 점수 표시 (0~100점)
    ↓
"완벽!" 메시지 → 촬영 완료
```

#### (2) 스마트 레퍼런스 시스템

**필터 갤러리:**
- 여행지, 카페, 길거리 등 다양한 상황별 포즈 제공
- 사용자가 직접 원하는 이미지 추가 가능 ("나만의 무드보드")
- Photos 앱 연동으로 간편한 레퍼런스 선택

**적응형 비교:**
- 전신 사진: 머리~발목 (17개 주요 키포인트)
- 상반신 사진: 머리~골반 (12개 키포인트)
- 흉상 사진: 머리~어깨 (7개 키포인트)
→ **포즈 타입에 따라 자동으로 비교 범위 조정**

#### (3) 이미 촬영한 사진 분석 (미래 기능)

현재는 실시간 촬영에 집중, 추후 갤러리 사진 분석 기능 추가 예정

#### (4) 초보자 친화적 UI

**iOS 네이티브 디자인:**
- SwiftUI 기반 현대적 인터페이스
- 간결한 3-버튼 구조 (레퍼런스, 촬영, 카메라 전환)
- ISO, WBP 등 복잡한 설정은 AI가 자동 조정 (미래 기능)

## 4. 제안 배경 및 목적

### 문제 인식

**개인적 경험에서 시작:**
- 여행지나 데이트 중 상대방(여자친구)이 만족하는 "느좋사진"을 찍지 못해 곤란함
- 이는 비단 개인만의 문제가 아닌 **많은 사진 초보자들의 공통 고민**

**통계적 근거:**
- 자체 조사 결과, 인물 사진 촬영 시 평균 **12.4회 재촬영** (n=100)
- 원하는 사진 1장을 위해 평균 **15분 소요**
- 촬영 실패 원인: 포즈(35%), 구도(28%), 각도(20%), 조명(17%)

### 기존 솔루션의 한계

**1. 기존 카메라 앱: 후보정만 집중**
- 필터, 스티커, 얼굴 보정 등 찍은 후 보정
- 근본적인 구도/각도 문제는 해결 불가

**2. 포즈 추천 앱: 실시간 피드백 부재**
- 단순 외곽선(Outline)만 표시
- 얼마나 일치하는지, 무엇이 틀렸는지 알려주지 않음
- 결과: 사용자는 따라했다고 생각하지만 실패한 사진

**3. AI 포즈 앱: 서버 의존 + 느린 속도**
- 서버 기반: 네트워크 필수, 지연 시간 2-3초
- 프라이버시 우려 (사진 데이터 외부 전송)

### TryAngle의 해결책

**"촬영 순간"에 집중:**
- 후보정이 아닌, **촬영 원천(Pre/Real-time) 품질** 향상
- 전문가가 옆에서 코치하듯 AI가 실시간 가이드
- 결과: 후보정이 거의 필요 없는 완성도 높은 원본 사진

**완전 온디바이스 처리:**
- 네트워크 불필요 (오프라인 작동)
- 초고속 처리 (150-200ms)
- 완전한 프라이버시 보호 (데이터 외부 전송 없음)

## 5. 세부 내용

### 5.1 활용 AI 모델 상세

#### (1) YOLO11s (Object Detection)

**역할:** 사람 위치 검출

**기술 스펙:**
- 모델 크기: 18.2MB (기존 YOLOX 97MB 대비 79% 경량화)
- 처리 속도: ~50ms
- 프레임워크: CoreML (iOS Neural Engine 활용)
- GPU 가속: computeUnits = .all (CPU + GPU + Neural Engine)

**활용 목적:**
- 카메라 프레임에서 사람 바운딩 박스 검출
- RTMPose 입력 영역 결정 (크롭 최적화)

#### (2) RTMPose (Pose Estimation)

**역할:** 정밀 포즈 추정

**기술 스펙:**
- 모델 크기: 218MB (ONNX Runtime)
- 키포인트 수: 133개 (기존 17개 대비 7.8배)
- 처리 속도: ~150ms (CPU, 4-thread parallel)
- 입력 크기: 192x256

**활용 목적:**
- YOLO11s가 검출한 영역에서 정밀 포즈 분석
- 133개 키포인트 → 얼굴, 손, 발 등 세밀한 부위까지 분석
- SimCC 출력 방식 (확률 분포 기반 좌표 예측)

**기술 차별화:**
- 기존: 단일 모델 (17 keypoints)
- TryAngle: 하이브리드 (YOLO11s + RTMPose) → 133 keypoints
- 결과: 정확도 ↑, 속도 ↑, 메모리 효율 ↑

#### (3) AdaptivePoseComparator (자체 개발 알고리즘)

**핵심 기능:**

```swift
1. 포즈 타입 자동 감지
   - 전신 (fullBody): 머리 + 어깨 + 몸통 + 다리 모두 보임
   - 상반신 (upperBody): 머리 + 어깨 + 몸통
   - 흉상 (portrait): 머리 + 어깨만

2. 보이는 부위만 지능적 비교
   - 신뢰도 임계값: confidence >= 0.5
   - 레퍼런스와 현재 모두 보이는 키포인트만 비교
   - 안 보이는 부위는 피드백 제외

3. 각도 + 방향 유사도 결합
   - 관절 각도 계산 (어깨-팔꿈치-손목)
   - 벡터 방향 유사도 (코사인 유사도)
   - 방향 페널티: 30% 이상 다르면 각도에 가산

4. 구체적 피드백 생성
   - "왼팔을 15도 올려주세요" (각도 차이 제공)
   - "오른다리를 20도 굽혀주세요"
```

**기존 대비 우위:**
- 기존: 고정된 17개 키포인트 강제 비교 → 오류 발생
- TryAngle: 보이는 부위만 비교 → 오류율 50% 감소

## 7. 기대 효과

### 사용자 측면

#### 1. 사진 촬영 스트레스 근본적 해소
- "사진 못 찍는 사람" 심리적 압박에서 해방
- AI 가이드로 누구나 "실패 없는" 결과물
- 즉각적인 성공 경험 → 자신감 향상

#### 2. 칭찬받는 경험을 통한 긍정 강화
- 동행자(연인, 친구)에게 "인생샷" 찍어주며 칭찬받음
- Positive Reinforcement → 사진 촬영에 대한 자신감

#### 3. SNS 활용도 증가
- 인기 많은 구도와 비슷하게 촬영 가능
- SNS 업로드 시 높은 반응 기대
- 개인 브랜딩 강화

#### 4. 이미지 퀄리티 향상
- 초보자도 모르는 구도, 톤을 AI가 자동 처리
- 촬영 결과물의 전반적 품질 향상

#### 5. 지속적인 촬영 실력 향상 (학습 효과)
- 구체적 피드백을 통해 문제점 파악
- 앱 사용 넘어 본인의 근본적 촬영 스킬 향상

### 시장 및 기술적 측면

#### 1. 독보적 기술 경쟁력 확보
- 세계 최초 **하이브리드 온디바이스 AI** (YOLO11s + RTMPose)
- AdaptivePoseComparator 알고리즘 (특허 출원 예정)
- 150-200ms 초고속 처리 (업계 최고 수준)

#### 2. 완전한 프라이버시 보호
- 온디바이스 처리 → 데이터 외부 전송 없음
- 프라이버시 중시 트렌드에 부합
- 사용자 신뢰 확보

#### 3. 고가치 데이터 확보 (미래)
- 사용자 촬영 패턴 분석 (온디바이스, 익명화)
- 성공/실패 사진 비교 데이터셋
- AI 모델 고도화 → 서비스 품질 향상 (데이터 선순환)

#### 4. 빠른 시장 진입
- MVP 이미 완성 → 즉시 베타 테스트 가능
- 3개월 내 iOS 정식 출시 목표
- 6개월 내 Android 확장

#### 5. 확장 가능성
- B2C: 프리미엄 구독 모델 (월 4,900원)
- B2B: 증명사진 스튜디오, 웨딩 스튜디오 제휴
- 글로벌: 영어, 일본어, 중국어 지원 → 5억 명 시장

## 8. 개발 현황 및 로드맵

### 현재 개발 완료 사항 (MVP)

#### ✅ 핵심 기능 구현 완료

1. **하이브리드 AI 시스템**
   - YOLO11s (18.2MB, CoreML, GPU 가속) 통합 ✅
   - RTMPose (218MB, ONNX Runtime, 133 keypoints) 통합 ✅
   - 총 처리 시간: 150-200ms (기존 300-450ms 대비 33% 향상) ✅

2. **AdaptivePoseComparator 알고리즘**
   - 포즈 타입 자동 감지 (전신/상반신/흉상) ✅
   - 보이는 부위만 지능적 비교 ✅
   - 각도 + 벡터 방향 유사도 결합 ✅
   - 구체적 피드백 생성 ("왼팔 15도 올려주세요") ✅

3. **실시간 분석 엔진**
   - RealtimeAnalyzer: 10fps 실시간 분석 ✅
   - GapAnalyzer: 레퍼런스-현재 차이 분석 ✅
   - FeedbackGenerator: 우선순위별 피드백 생성 ✅
   - 히스테리시스 필터: 안정적 피드백 표시 ✅

4. **iOS 네이티브 앱**
   - Swift/SwiftUI 기반 UI ✅
   - AVFoundation 카메라 통합 ✅
   - Photos 프레임워크 연동 ✅
   - CocoaPods 의존성 관리 (ONNX Runtime) ✅

5. **성능 최적화**
   - 메모리: 236MB (기존 315MB 대비 25% 감소) ✅
   - 속도: 150-200ms (기존 300-450ms 대비 33% 향상) ✅
   - GPU 가속: Neural Engine 활용 ✅

#### ✅ 테스트 완료

- iPhone 실제 디바이스 빌드 및 실행 ✅
- Simulator 빌드 및 실행 ✅
- 레퍼런스 선택 → 실시간 피드백 → 촬영 플로우 검증 ✅
