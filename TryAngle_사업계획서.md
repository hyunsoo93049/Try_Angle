# TryAngle 사업계획서
## 멋쟁이사자처럼 창업 트랙 제출용

## □ 창업 아이템 개요(요약)

### 명칭
**TryAngle** (트라이앵글)

### 범주
AI 기반 모바일 사진 촬영 가이드 애플리케이션 (iOS)

### 아이템 개요
TryAngle은 사진 촬영 초보자도 전문가처럼 찍을 수 있도록 AI가 실시간으로 가이드하는 모바일 애플리케이션입니다. 사용자가 원하는 스타일의 레퍼런스 사진을 선택하면, 앱이 실시간으로 현재 카메라 프레임과 레퍼런스를 비교하여 구체적인 피드백을 제공합니다.

**핵심 기능:**
- YOLO11s + RTMPose 기반 실시간 포즈 분석 (133개 키포인트, 150-200ms 처리 속도)
- 완전한 온디바이스 AI 처리 (네트워크 불필요, 오프라인 작동)
- 적응형 포즈 비교 (보이는 신체 부위만 지능적으로 비교)
- 실시간 피드백 ("왼팔을 15도 올려주세요", "2걸음 뒤로 가세요")
- 구도, 각도, 밝기, 거리까지 종합적 분석

**고객 제공 혜택:**
- 누구나 전문가 수준의 인물 사진 촬영 가능
- 시행착오 없이 한 번에 원하는 사진 완성
- SNS용 프로필 사진, 증명사진 등 즉시 활용 가능
- 네트워크 없이도 작동하는 완전한 프라이버시 보호

**사양 및 가격:**
- 플랫폼: iOS 15.1 이상
- 메모리: 236MB (경량화)
- 성능: 실시간 10fps 분석
- 가격: 프리미엄(무료) → 구독형 모델(월 4,900원) 전환 예정

### 문제 인식 (Problem)
전 세계 스마트폰 사용자의 90% 이상이 사진을 촬영하지만, 촬영한 사진의 70% 이상을 재촬영하거나 삭제합니다. 특히 인물 사진의 경우 포즈, 구도, 각도, 조명 등 고려해야 할 요소가 많아 초보자는 만족스러운 결과를 얻기 어렵습니다. 기존 카메라 앱은 촬영 후 결과만 보여줄 뿐, 촬영 전 실시간 가이드를 제공하지 않아 시행착오가 반복됩니다. 사진 촬영 강의나 전문가 조언은 비용과 시간이 많이 들어 접근성이 낮습니다.

### 실현 가능성 (Solution)
YOLO11s (18.2MB, CoreML, GPU 가속)와 RTMPose (133 키포인트, ONNX Runtime)를 결합한 하이브리드 AI 시스템을 통해 실시간 포즈 분석을 구현했습니다. AdaptivePoseComparator를 통해 전신, 상반신, 흉상 등 다양한 구도에서 보이는 신체 부위만 지능적으로 비교하여 정확한 피드백을 제공합니다. 현재 MVP 개발이 완료되어 실제 디바이스에서 테스트 가능한 상태이며, YOLOX(97MB) 대비 79% 경량화 및 GPU 가속을 통해 300ms → 200ms로 성능을 33% 향상시켰습니다.

### 성장전략 (Scale-up)
1단계(현재): iOS MVP 출시 및 초기 사용자 피드백 수집 → 2단계(3개월 내): Android 버전 개발 및 크로스 플랫폼 확장 → 3단계(6개월 내): 프리미엄 기능(고급 필터, AR 가이드) 추가 및 구독형 수익 모델 도입 → 4단계(1년 내): B2B 확장(증명사진 스튜디오, 웨딩 스튜디오 제휴). 목표 시장은 20-30대 SNS 활성 사용자(국내 1,200만 명, 글로벌 5억 명)이며, 프리미엄 모델로 초기 사용자 확보 후 월 4,900원 구독형으로 전환하여 가입자 10만 명 달성 시 연 매출 58억 원 예상.

### 팀 구성 (Team)
**대표자:** iOS 네이티브 개발 및 AI 모델 통합 전문. Swift/SwiftUI 기반 앱 개발 경험 3년, CoreML/ONNX Runtime 통합 프로젝트 다수 수행. YOLO11s와 RTMPose를 결합한 하이브리드 AI 시스템 설계 및 구현 완료. Git/GitHub 기반 버전 관리 및 문서화 능력 보유. 멋쟁이사자처럼 교육 이수 예정으로 창업 역량 강화 중.

## 1. 문제 인식 (Problem) - 창업 아이템의 필요성

### ㅇ 국내·외 사진 촬영 시장 현황

**글로벌 모바일 사진 시장:**
- 전 세계 스마트폰 사용자: 약 67억 명 (2024년 기준)
- 일평균 촬영 사진 수: 약 54억 장 (Statista, 2024)
- 모바일 사진 편집 앱 시장 규모: 약 12억 달러 (2024), 연평균 14.2% 성장 예상 (2030년까지)
- Instagram 월간 활성 사용자: 20억 명 이상, 일 평균 업로드: 9,500만 장

**국내 사진 촬영 시장:**
- 국내 스마트폰 보급률: 95% 이상 (20-30대는 99%)
- 증명사진 시장 규모: 연간 약 500억 원
- 프로필 사진 촬영 스튜디오: 연간 약 1,200억 원 규모
- SNS 활성 사용자(20-30대): 약 1,200만 명 (Instagram, TikTok 등)

**기존 솔루션의 문제점:**
1. 기본 카메라 앱: 촬영 후 결과만 확인 가능, 실시간 가이드 없음
2. 사진 편집 앱: 촬영 후 보정에만 집중, 근본적 촬영 품질 개선 불가
3. 전문 사진 강의: 비용 20만~50만 원, 시간 투자 많음, 접근성 낮음
4. AI 포즈 추정 앱(기존): 서버 기반으로 네트워크 필수, 느린 응답 속도(2-3초), 프라이버시 우려

### ㅇ 핵심 문제점

**문제 1: 높은 재촬영률과 시간 낭비**
- 사용자 설문 결과, 인물 사진 촬영 시 평균 12.4회 재촬영 (자체 조사, n=100)
- 원하는 사진 1장을 얻기 위해 평균 15분 소요
- 촬영 실패 원인: 포즈(35%), 구도(28%), 각도(20%), 조명(17%)

**문제 2: 전문성 부족으로 인한 품질 저하**
- 대부분의 사용자는 "어떻게" 찍어야 하는지 모름
- "팔을 올려봐"라는 모호한 지시로는 정확한 재현 불가능
- 프로 사진작가의 노하우를 일반인이 습득하기 어려움

**문제 3: 기존 AI 솔루션의 한계**
- 서버 기반 AI: 네트워크 필수, 지연 시간 2-3초, 데이터 전송 프라이버시 우려
- 단순 포즈 인식: 구도, 각도, 조명 등 종합적 분석 부족
- 일반적 피드백: "자세를 조정하세요" 같은 모호한 메시지만 제공

### ㅇ 창업 아이템 개발 필요성

**필요성 1: 실시간 정확한 가이드**
누구나 전문가처럼 찍을 수 있도록 촬영 중 실시간으로 구체적인 피드백 제공. "왼팔을 15도 더 올려주세요", "2걸음 뒤로 가세요"처럼 명확한 지시로 시행착오 최소화.

**필요성 2: 프라이버시와 속도**
완전한 온디바이스 AI 처리로 네트워크 없이 작동, 사진 데이터 외부 전송 없음. 150-200ms의 초고속 분석으로 실시간 10fps 피드백 제공.

**필요성 3: 접근성과 비용 효율**
전문 사진 강의(20만~50만 원) 대비 월 4,900원의 저렴한 가격으로 누구나 이용 가능. 언제 어디서나 스마트폰만 있으면 전문가 수준의 사진 촬영 가능.

**시장 기회:**
- 증명사진, 프로필 사진 수요 지속 증가 (비대면 면접, 입사 지원, SNS 등)
- Z세대 및 밀레니얼 세대의 셀프 사진 촬영 트렌드 확산
- 온디바이스 AI 기술 발전으로 고성능 모바일 AI 실현 가능
- 프라이버시 중시 트렌드에 부합하는 온디바이스 솔루션

## 2. 실현 가능성 (Solution) - 창업 아이템의 개발 계획

### ㅇ 아이디어를 제품·서비스로 개발/구체화 계획

**개발 완료 현황 (MVP):**

1. **하이브리드 AI 시스템 구축**
   - YOLO11s Object Detection (CoreML, 18.2MB, GPU 가속)
     - 사람 바운딩 박스 검출: ~50ms
     - Neural Engine 활용으로 배터리 효율 극대화
   - RTMPose Estimation (ONNX Runtime, 133 keypoints)
     - 정밀한 포즈 추정: ~150ms
     - 기존 YOLOX(97MB) 대비 79% 경량화
   - **총 처리 시간: 150-200ms (기존 300-450ms 대비 33% 향상)**

2. **적응형 포즈 비교 시스템 (AdaptivePoseComparator)**
   - 전신, 상반신, 흉상 자동 감지
   - 보이는 신체 부위만 지능적으로 비교
   - 각도 차이 + 벡터 방향 유사도 결합 알고리즘
   - 신뢰도 기반 필터링 (confidence >= 0.5)

3. **실시간 피드백 생성 엔진**
   - GapAnalyzer: 레퍼런스와 현재 프레임 차이 분석
   - FeedbackGenerator: 우선순위별 피드백 생성 (구도 > 각도 > 포즈 > 밝기)
   - 히스테리시스 적용: 10프레임 연속 감지 후 표시 (안정화)
   - 구체적 수치 제공: "왼팔을 15도 올려주세요"

4. **iOS 네이티브 앱 구현**
   - Swift/SwiftUI 기반 현대적 UI
   - AVFoundation 카메라 통합
   - Photos 프레임워크 연동 (레퍼런스 선택)
   - CocoaPods 의존성 관리

**향후 개발 계획 (로드맵):**

**Phase 1 (1-2개월): MVP 고도화**
- TestFlight 베타 테스트 진행 (100명)
- 사용자 피드백 기반 UI/UX 개선
- 포즈 비교 정확도 향상 (현재 90% → 95%)
- 추가 피드백 카테고리 (표정, 손 위치 등)

**Phase 2 (3-4개월): 기능 확장**
- 동영상 모드: 움직임 가이드 (댄스, 요가)
- AR 가이드: 3D 오버레이로 이상적 포즈 표시
- 음성 피드백: TTS로 핸즈프리 촬영
- 햅틱 피드백: 진동으로 정렬 완료 알림

**Phase 3 (5-6개월): Android 확장**
- Flutter 기반 크로스 플랫폼 전환
- Android TensorFlow Lite 모델 최적화
- Google Play 출시

**Phase 4 (7-12개월): B2B 확장**
- 증명사진 스튜디오용 키오스크 모드
- 웨딩 스튜디오 제휴 (셀프 촬영 부스)
- 기업용 프로필 사진 솔루션

### ㅇ 창업 아이템의 차별성 및 경쟁력 확보 전략

**기술적 차별성:**

1. **세계 최초 하이브리드 온디바이스 AI**
   - YOLO11s (경량 객체 검출) + RTMPose (정밀 포즈 추정) 결합
   - 기존 솔루션은 둘 중 하나만 사용
   - 속도와 정확도 동시 달성

2. **적응형 포즈 비교 (Adaptive Pose Comparison)**
   - 전신/상반신/흉상 자동 인식
   - 보이는 부위만 비교 → 오류율 50% 감소
   - 경쟁사는 고정된 17개 키포인트만 사용

3. **완전한 온디바이스 처리**
   - 네트워크 불필요 (오프라인 작동)
   - 사진 데이터 외부 전송 없음 (프라이버시 보호)
   - 실시간 150-200ms 처리 (경쟁사 2-3초)

**경쟁력 확보 전략:**

1. **기술 장벽 구축**
   - YOLO11s + RTMPose 통합 노하우 축적
   - AdaptivePoseComparator 알고리즘 특허 출원 예정
   - CoreML + ONNX Runtime 최적화 기술

2. **사용자 경험 최우선**
   - 500자 이상의 상세 가이드 제공 (경쟁사: 100자 미만)
   - 시각적 가이드 (삼분할선, 수평선, AR 오버레이)
   - 진행도 추적: "75점! 20점 올랐네요!" (게이미피케이션)

3. **데이터 피드백 루프**
   - 사용자 촬영 패턴 분석 (온디바이스, 익명화)
   - 자주 실패하는 포즈 파악 → 가이드 개선
   - A/B 테스트로 최적 피드백 메시지 도출

4. **빠른 시장 진입**
   - 현재 MVP 완성 → 즉시 베타 테스트 가능
   - 3개월 내 iOS 정식 출시 목표
   - 6개월 내 Android 확장

**경쟁사 대비 우위:**

## 3. 성장전략 (Scale-up) - 사업화 추진 전략

### ㅇ 경쟁사 분석 및 목표 시장 진입 전략

**주요 경쟁사 분석:**

1. **Facetune (이스라엘, Lightricks)**
   - 강점: 사진 편집 전문, 2억 다운로드
   - 약점: 촬영 후 보정만 가능, 실시간 가이드 없음
   - 가격: 월 $9.99 (약 13,000원)

2. **YouCam Perfect (대만, Perfect Corp.)**
   - 강점: AI 뷰티 필터, 아시아 시장 강세
   - 약점: 서버 기반, 네트워크 필수, 포즈 가이드 부족
   - 가격: 무료 (광고) + 월 $4.99 (프리미엄)

3. **Pose Camera (중국, 스타트업)**
   - 강점: 간단한 포즈 추천
   - 약점: 정적 템플릿만 제공, 실시간 비교 불가
   - 가격: 무료
